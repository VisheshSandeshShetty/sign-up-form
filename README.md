# sign-up-form

Form created with image on left hand side and form with some text above it. 
Form includes mandatory fields and also phone number allows only 10 digit numbers.
Mandatory fields are marked with asterisk(*).
E-mail must be of the form user@example.com.
Password is unrestricted however when you enter it, it is hidden using dots.

    Clicking on 'SUBMIT' leads to a nice present!
